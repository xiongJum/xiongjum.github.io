Ⅰ，理解
  • 指对不同类型的变量进行操作，根据对象（或类）类型的不同而表现出不同的行为
Ⅱ，常用的多态
  • 不同的对象（或类）调用‘’+”，“+”会做出不同的响应
Ⅲ，对象所属的类之间有继承关系
Ⅳ，对象所属的类之间没有继承关系
  • 调用同一个函数，传入不同的参数，以达到不同的功能
