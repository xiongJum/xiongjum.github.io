子查询
  • 嵌套在另一个查询中的查询
    • 放在括号内的查询成为子查询，也被称为内部查询或内部选择
    • 包含子查询的查询称为外部查询或外部选择
表连接
  • 内连接
    • 关键词 inner join … on
    • 只会关联并展示两张表中都存在的数据
  • 左连接
    • 关键词 left join … on
    • 以主表为主，展示所有主表的行数据，若关联的表没有匹配则展示null
  • 右连接
    • 关键词 right join … on
    • 和左连接相反，以关联的表为主，会展示所有关联表中的行数据
  • 全外连接
    • 关键词 full join …on
    • 展示全部的行数
  • 多表连接
    • 超过两张表的表连接
